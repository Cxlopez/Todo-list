# Todo List

This is just a simple Todo list that lets you `add`, `edit` & `delete` your tasks for the day.

## Setup
- Fork and clone this repo
- Install dependencies with `npm install`
- Run server with `npm start`


## Dependencies 
- React
- React-dom
- React-scripts
- React-icons
